# El-laberinto-maestro

Repositorio para el Challenge 2 - El laberinto maestro.

## Generación del laberinto.

El laberinto fue generado con un algoritmo DFS con saltos de 2 en 2 sin alcanzar los muros externos.

## Algoritmo de resolución.

El algoritmo usado para resolver el laberinto fue el BFS porque es el que detecta más fácilmente la ruta más corta, 
aunque sea un proceso mas largo.

## Mejoras.

Añadiria multiples rutas en laberintos más grandes.

## Tiempo de resolución.

Me sorprendio que no esperaba que BFS sea tan bueno resolviendo laberintos puesto que uno grande de 30 x 30 lo resuelve entre 0.12 y 0.14 milisegundos. Esto demuestra que el algoritmo es eficiente incluso para laberintos relativamente grandes.
